# AlastackAI
Plantilla automatizada (Boilerplate) para desplegar entornos de desarrollo Multi-Agente (Agents Team Lite) con IA. Aísla dominios en Git Worktrees, inyecta Clean Architecture e infraestructura Zero Trust, y configura perfiles especializados (Orchestrator, Implementer, DevOps) con un CLI unificado (gga). Preparado para Linux/macOS y Windows.
